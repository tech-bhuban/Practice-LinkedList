# Singly Linked List Implementation in C++

A clean and efficient implementation of a singly linked list data structure in C++ with basic insertion operations.

## 📋 Features

- **Node-based implementation** - Each node contains data and a pointer to the next node
- **Insertion operations**:
  - Insert at head (beginning)
  - Insert at end (tail)
  - Insert at specific position
- **Memory management** - Proper handling of edge cases and memory allocation
- **Error handling** - Boundary checking for position-based insertion

## 🚀 Getting Started

### Prerequisites

- C++ compiler (g++, clang++, or MSVC)
- C++11 or later

### Compilation

```bash
g++ -o linkedlist linkedlist.cpp
